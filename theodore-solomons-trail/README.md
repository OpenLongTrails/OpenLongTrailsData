Method used to obtain this route data:

1. Load https://overpass-turbo.eu/
2. Enter the following query:
    [out:json][timeout:60];
    rel(5549623);
    (._;>>;);
    out;
3. Select "Run"
4. Select "Export"
5. Select "GeoJSON" and "GPX"
