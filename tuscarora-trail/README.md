1. Load https://overpass-turbo.eu/
2. Enter the following query:
    [out:json][timeout:60];
    rel(10449263);
    (._;>>;);
    out;
3. Select "Run"
4. Select "Export"
5. Select "GeoJSON" and "GPX"
